Connect from c9.io to git:

Config
    git remote add upstream git@github.com:freak4fun/moneybox.git // mappen von Git Repository auf den alias upstream
    
Checkout
    git pull moneybox main // Änderungen vom Git Repository holen
    
Commit
    git add <filename> // Datei zum Git Repository hinzufügen
    git add . // Alle geänderten und neuen Dateien zum commit hinzufügen
    git commit -am 'kommentar' // alles kommiten
    git push upstream main // Änderungen ans Git Repository schicken
